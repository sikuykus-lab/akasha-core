# AKASHA lifecycle

Brain: `~/.akash/config.local`. Кубик = нейрон, weave = активация подсети.

## На задачу

1. `prepare "<задача>"` → weave.brief.md
2. `read-weave` → assembly
3. `read-skill` якорь (+ glue)
4. Склейка entrypoints → новое решение
5. `create-skill` + `## Назначение` + `sync`

Не листать skills/. Карта: `read-map`.

## Hooks

- sessionStart → pull
- stop → sync
